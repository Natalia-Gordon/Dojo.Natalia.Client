# Audio Files Directory

Place audio files for article readings here.

## Current Audio Files:

- `zen-today-dani-waxman.mp3` - Audio reading of "הזן היום" article by דני וקסמן

## Supported Formats:

- MP3 (recommended for best browser compatibility)
- OGG
- WAV (not recommended due to large file size)

## File Naming Convention:

Use descriptive names like: `[article-slug]-[author-name].mp3`

Example: `zen-today-dani-waxman.mp3`
